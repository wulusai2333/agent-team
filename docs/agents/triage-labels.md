# Triage 标签

技能使用五个标准 triage 角色。本文件将这些角色映射到仓库实际使用的标签字符串。

| 技能中的标签名        | 仓库实际标签          | 含义                     |
| -------------------- | -------------------- | ------------------------ |
| `needs-triage`       | `needs-triage`       | 维护者需要评估此 Issue     |
| `needs-info`         | `needs-info`         | 等待提问者补充更多信息      |
| `ready-for-agent`    | `ready-for-agent`    | 描述清晰，Agent 可独立处理  |
| `ready-for-human`    | `ready-for-human`    | 需要人类开发者实现          |
| `wontfix`            | `wontfix`            | 不会处理                  |

当技能提到某个角色时（如"打上 AFK 就绪的 triage 标签"），使用本表对应的标签字符串。

如需调整，直接修改右侧列的标签名即可。
